# Task 2: CLI Expense Tracker

## Description
This Python application acts as a command-line Expense Tracker. It prompts users to enter expense amounts, aggregates them, and provides a running total of their expenses until they choose to exit.

## Features
- **Running Total**: Instantly updates and displays the total amount spent after every new expense entry.
- **Robust Input Validation**:
  - Handles non-numeric inputs (strings, letters) gracefully by asking the user to enter a valid number.
  - Prevents negative expense values from being added and alerts the user.
- **Clear Exit Option**: Typing `quit` ends the loop and displays the final total spent.

## How to Run the Code
1. Open your terminal or command prompt.
2. Navigate to the folder containing `week2.py`.
3. Run the script using Python:
   ```bash
   python week2.py
   ```
